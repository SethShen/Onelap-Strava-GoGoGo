# 顽爪爪同步 - 操作指导

## 📱 功能介绍

**顽爪爪同步** 是一款将 OneLap（顽鹿）骑行 FIT 文件同步到 Strava 和行者平台的工具。

### 主要特性
- 从 OneLap 下载骑行 FIT 文件
- 支持 GCJ-02 → WGS84 坐标转换
- 同时上传至 Strava 和行者
- 智能去重（按平台独立判断）
- 同步历史记录查询

---

## 🚀 首次使用准备

### 1. 安装 App
安装 Android APK 后即可使用。

### 2. 准备账号

| 平台 | 必需项 | 说明 |
|------|--------|------|
| **OneLap** | 用户名 + 密码 | 必须，用于下载 FIT 文件 |
| **Strava** | API 凭证 + OAuth 授权 | 必须，技术门槛稍高 |
| **行者** | 用户名 + 密码 | 可选，按需开启 |

## ⚙️ 设置步骤

打开 App → 点击右上角 **⚙️ 设置图标** 进入设置页面。

### Step 1：填写 OneLap 账号

在设置页面上半部分：

```
OneLap 账号
  ├── OneLap 用户名：你的顽鹿用户名
  └── OneLap 密码：你的顽鹿密码
```

点击「保存 OneLap 账号」验证登录。

### Step 2：配置同步设置

```
同步设置
  ├── 上传前将 GCJ-02 转为 WGS84：开关
  │   └── 仅在轨迹明显偏移时开启（国内地图常用 GCJ-02 坐标系）
  └── 同步最近几天：默认 3
      └── 控制每次同步拉取的活动范围
```

点击「保存同步设置」。

### Step 3：设置上传平台

```
上传设置
  ├── 上传到 Strava：默认开启
  └── 上传到 行者：默认关闭（按需开启）
```

点击「保存上传设置」。

### Step 4：授权 Strava ⚠️ 重要

Strava 需要 OAuth 授权才能上传，需要你创建自己的 API 应用：

**4.1 创建 Strava API 应用**
1. 访问 https://www.strava.com/settings/api
2. 点击「Create An Application」
3. 填写信息：
   - **Application Name**：随便填，如 `My Sync App`
   - **Category**：选择 `Cycling`
   - **Club**：选 None
   - **Website**：填 `http://localhost`
   - **Authorization Callback Domain**：填 `localhost`
4. 创建后复制 **Client ID** 和 **Client Secret**

**4.2 在 App 中授权**
```
Strava 凭证
  ├── Strava Client ID：粘贴
  ├── Strava Client Secret：粘贴
  └── 点击「授权 Strava」按钮
```

授权成功后会跳回设置页，Access Token、Refresh Token 和 Expires At 将自动填入。

### Step 5：填写行者账号（如需）

```
行者 凭证
  ├── 行者 用户名：你的行者账号
  └── 行者 密码：你的行者密码
```

点击「登录 行者」验证。

## 🔄 同步操作流程

### 正常同步

1. 返回 App 首页
2. 点击 **「立即同步」** 按钮
3. 等待同步完成，会显示结果弹窗：
   - 成功数量
   - 失败数量及原因（如有）
4. 可选择复制失败详细信息

### 查看同步记录

**首页 Banner 列表**（同步按钮下方）：
- 显示最近 6 条同步记录
- 每条记录显示：时间、获取条数、通过条数、需同步条数、平台统计
- 左滑可删除单条记录
- 点击展开详情弹窗

**完整历史页面**：
- 点击首页右上角 **🕐 历史图标** 进入
- 支持按日期范围筛选
- 支持按平台/状态筛选（Strava / 行者 / 成功 / 失败 / 已同步）
- 显示总记录数
- 每条记录展示：日期标签、距离、爬升、文件名、各平台同步状态

## ❓ 常见问题

### Q: 同步失败，提示"请先在设置中填写 OneLap 凭证"
**A:** 设置 → OneLap 账号 → 填写用户名密码 → 点击「保存 OneLap 账号」

### Q: Strava 授权失败
**A:** 
1. 确认 Client ID 和 Client Secret 填写正确
2. 确认 Authorization Callback Domain 填写为 `localhost`
3. 点击「授权 Strava」按钮后，在打开的页面完成授权

### Q: 上传后 Strava 显示轨迹偏移
**A:** 设置 → 同步设置 → 开启「上传前将 GCJ-02 转为 WGS84」

### Q: 同步提示"没有要同步的啦"
**A:** 正常现象，说明最近几天的活动都已经同步过了。可调整「同步最近几天」数值。

### Q: 想重新同步某个活动
**A:** 当前版本不支持手动重新同步特定活动，需等待该活动在同步范围内时自动处理。
